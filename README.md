# Dois Websites

- **About Me** - Um site pessoal de apresentação com múltiplas páginas, estilo customizado e mídia incorporada.
- **Querido Blog** - Um site estilo blog com múltiplas páginas, estilo customizado e mídia incorporada também.

## Por que dois?

- Após alguns exercícios, acabei criando a estrutura do Querido Blog, que decidi por aprimorar conforme o projeto nesta reta final.
- Retomando exercícios do início do módulo, também iniciei uma repaginação do exercício de apresentação com foto, link do linkedin, etc.

## Tecnologias

- HTML
- CSS (Flexbox, Grid, Media Queries, Custom Fonts)
- Embeds do Apple Music
